Getting Started

Learn the basic structure of a SPIKE Prime Python program and write your first "Hello, World!" example.

Basic Program Structure:

# Import required modules
from hub import light_matrix

# Simple program to display text
light_matrix.write('Hello, World!')

Using Async Functions (Recommended):

import runloop
from hub import light_matrix

async def main():
    light_matrix.write('Hello from async!')
    # Add more code here

runloop.run(main())

Why use async?

Async functions allow multiple operations to run at the same time, like moving a motor while checking sensors. This is essential for responsive robot behaviour.

Pro Tip:

Always use the async structure for any program that controls motors or reads sensors. It makes your robot much more responsive and professional.

Moving Forward

Control a single motor or create coordinated movement using motor pairs for driving forward.

Hardware needed: Connect motors to ports A and B for a basic drivetrain

Single Motor Control:

import motor
import runloop
from hub import port

async def main():
    # Run motor for 360 degrees (1 rotation) at 720 degrees/sec
    await motor.run_for_degrees(port.A, 360, 720)

runloop.run(main())

Motor Pair for Driving:

import motor_pair
import runloop
from hub import port

async def main():
    # Pair motors on ports A and B
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    # Move straight forward for 1000ms at 360 degrees/sec
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, 0, velocity=360)
    
    # Move forward for specific distance (degrees)
    await motor_pair.move_for_degrees(motor_pair.PAIR_1, 720, 0, velocity=360)

runloop.run(main())

Motor Parameters:

  • degrees: How far to move (360 = one full rotation)
  • velocity: Speed in degrees per second (360 = slow, 1000 = fast)
  • steering: Direction (-100 = left, 0 = straight, 100 = right)

Pro Tip:

Use await before motor commands to wait for completion, or omit it to start the action and immediately move to the next line.

Turning & Steering

Make your robot turn, pivot, and steer with precision using motor pairs and tank control.

Steering with Motor Pairs:

import motor_pair
import runloop
from hub import port

async def main():
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    # Turn right while moving forward
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, 50, velocity=360)
    
    # Turn left while moving forward  
    await motor_pair.move_for_time(motor_pair.PAIR_1, 1000, -50, velocity=360)
    
    # Sharp right turn (pivot)
    await motor_pair.move_for_time(motor_pair.PAIR_1, 500, 100, velocity=360)

runloop.run(main())

Tank Control (Individual Motor Speeds):

import motor_pair
import runloop
from hub import port

async def main():
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    # Move straight (both motors same speed)
    await motor_pair.move_tank_for_time(motor_pair.PAIR_1, 500, 500, 1000)
    
    # Turn right (left motor faster)
    await motor_pair.move_tank_for_time(motor_pair.PAIR_1, 800, 200, 1000)
    
    # Spin in place (motors opposite directions)
    await motor_pair.move_tank_for_time(motor_pair.PAIR_1, 500, -500, 1000)

runloop.run(main())

Steering Values:

  • -100: Sharp left turn
  • -50: Gentle left curve
  • 0: Straight forward
  • 50: Gentle right curve
  • 100: Sharp right turn

Pro Tip:

Use tank control for precise movements and steering for smooth curves. Experiment with different values to find what works best for your robot!

Using the Colour Sensor

Detect colours and light intensity to make your robot respond to its environment.

Hardware needed: Connect Colour Sensor to any port (A-F)

Basic Colour Detection:

import color_sensor
import color
import runloop
from hub import port

async def main():
    while True:
        detected_color = color_sensor.color(port.A)
        
        if detected_color == color.RED:
            print("Red detected!")
        elif detected_color == color.BLUE:
            print("Blue detected!")
        elif detected_color == color.GREEN:
            print("Green detected!")
        else:
            print("Unknown color")
            
        await runloop.sleep_ms(100)

runloop.run(main())

Light Intensity (Reflection):

import color_sensor
import runloop
from hub import port

async def main():
    while True:
        # Get light reflection (0-100%)
        reflection = color_sensor.reflection(port.A)
        print("Light reflection:", reflection, "%")
        
        if reflection > 70:
            print("Very bright surface")
        elif reflection < 30:
            print("Dark surface")
        else:
            print("Medium brightness")
            
        await runloop.sleep_ms(200)

runloop.run(main())

Detected Colours:

The sensor can detect: Red, Green, Blue, Yellow, Magenta, Orange, Azure, Black, White, and Unknown

Pro Tip:

For line following, use reflection values rather than colours. Dark lines typically have reflection values below 30%.

Using the Distance Sensor

Measure distances and create light displays with the ultrasonic Distance Sensor.

Hardware needed: Connect Distance Sensor to any port (A-F)

Measuring Distance:

import distance_sensor
import runloop
from hub import port

async def main():
    while True:
        # Get distance in millimeters
        distance = distance_sensor.distance(port.A)
        
        if distance == -1:
            print("No object detected")
        else:
            print("Distance:", distance, "mm")
            
        if distance < 100 and distance > 0:
            print("Object very close!")
        elif distance < 300 and distance > 0:
            print("Object nearby")
            
        await runloop.sleep_ms(100)

runloop.run(main())

Obstacle Avoidance:

import distance_sensor
import motor_pair
import runloop
from hub import port

async def main():
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    while True:
        distance = distance_sensor.distance(port.C)
        
        if distance < 200 and distance > 0:
            # Obstacle detected - turn right
            await motor_pair.move_for_time(motor_pair.PAIR_1, 500, 100, velocity=360)
        else:
            # No obstacle - move forward
            motor_pair.move(motor_pair.PAIR_1, 0, velocity=360)
            
        await runloop.sleep_ms(50)

runloop.run(main())

Distance Range:

  • Range: 30mm to 2500mm (3cm to 2.5m)
  • Returns -1: When no object is detected
  • Works best: With flat, solid surfaces

Pro Tip:

Always check if the distance is greater than 0 before using the value. A return of -1 means no valid reading.

Using the Force Sensor

Detect touch, pressure, and force to create interactive robot behaviours.

Hardware needed: Connect Force Sensor to any port (A-F)

Detecting Button Press:

import force_sensor
import runloop
from hub import port, light_matrix

async def main():
    while True:
        if force_sensor.pressed(port.A):
            light_matrix.show_image(light_matrix.IMAGE_HAPPY)
            print("Force sensor pressed!")
        else:
            light_matrix.show_image(light_matrix.IMAGE_ASLEEP)
            
        await runloop.sleep_ms(100)

runloop.run(main())

Measuring Force Amount:

import force_sensor
import motor
import runloop
from hub import port

async def main():
    while True:
        # Get force in decinewtons (0-100)
        force = force_sensor.force(port.A)
        print("Force:", force, "dN")
        
        # Use force to control motor speed
        # Multiply by 10 to get good motor speeds
        motor_speed = force * 10
        
        if motor_speed > 50:
            motor.run(port.B, motor_speed)
        else:
            motor.stop(port.B)
            
        await runloop.sleep_ms(50)

runloop.run(main())

Force Values:

  • Range: 0-100 decinewtons (dN)
  • Pressed: Returns True/False for any pressure
  • Light touch: ~10-20 dN
  • Firm press: ~50-80 dN

Pro Tip:

Use pressed() for simple button detection, and force() when you need to know how hard something is being pressed.

Playing Sounds

Create beeps, tones, and sound effects to give your robot audio feedback.

Basic Beeps:

from hub import sound
import runloop

async def main():
    # Simple beep (440Hz for 500ms at volume 100)
    await sound.beep(440, 500, 100)
    
    # High pitch beep
    await sound.beep(880, 300, 100)
    
    # Low pitch beep
    await sound.beep(220, 800, 100)
    
    # Very short beep for button feedback
    await sound.beep(600, 100, 80)

runloop.run(main())

Sound with Sensor Feedback:

import distance_sensor
from hub import sound, port
import runloop

async def main():
    while True:
        distance = distance_sensor.distance(port.A)
        
        if distance > 0 and distance < 300:
            # Higher pitch for closer objects
            frequency = 2000 - (distance * 5)  # Closer = higher pitch
            await sound.beep(frequency, 100, 50)
            
        await runloop.sleep_ms(100)

runloop.run(main())

Musical Sequences:

from hub import sound
import runloop

async def main():
    # Musical notes (approximate frequencies)
    C4 = 262
    D4 = 294
    E4 = 330
    F4 = 349
    G4 = 392
    
    # Play a simple melody
    notes = [C4, D4, E4, F4, G4]
    
    for note in notes:
        await sound.beep(note, 400, 80)

Sound Parameters:

  • Frequency: 200-2000 Hz (higher = more high-pitched)
  • Duration: Time in milliseconds
  • Volume: 0-100 (0 = silent, 100 = loudest)

Pro Tip:

Use different pitches for different robot states: high pitch for success, low pitch for errors, ascending notes for startup!

Menu System with Buttons:

from hub import button, light_matrix, sound
import runloop

async def main():
    menu_option = 1
    max_options = 3
    
    while True:
        # Show current menu option
        if menu_option == 1:
            light_matrix.show_image(light_matrix.IMAGE_HAPPY)
        elif menu_option == 2:
            light_matrix.show_image(light_matrix.IMAGE_HEART)
        elif menu_option == 3:
            light_matrix.show_image(light_matrix.IMAGE_YES)
        
        # Check buttons
        if button.pressed(button.LEFT):
            menu_option = menu_option - 1
            if menu_option < 1:
                menu_option = max_options
            await sound.beep(400, 100, 50)
            await runloop.sleep_ms(300)  # Prevent rapid scrolling
            
        elif button.pressed(button.RIGHT):
            menu_option = menu_option + 1
            if menu_option > max_options:
                menu_option = 1
            await sound.beep(600, 100, 50)
            await runloop.sleep_ms(300)  # Prevent rapid scrolling
            
        await runloop.sleep_ms(50)

runloop.run(main())

Button Functions:

  • button.LEFT: Left button (next to power button)
  • button.RIGHT: Right button (next to power button)
  • pressed(): Returns duration in milliseconds (0 = not pressed)

Pro Tip:

Add a small delay after detecting button presses to prevent rapid repeated triggers. Use different beep tones for button feedback!

Loops & Repetition

Use loops to repeat actions and create continuous robot behaviours.

Forever Loop (while True):

from hub import light_matrix
import runloop
import random

async def main():
    # This loop runs forever
    while True:
        # Show random emotion
        emotions = [
            light_matrix.IMAGE_HAPPY,
            light_matrix.IMAGE_SAD,
            light_matrix.IMAGE_SILLY,
            light_matrix.IMAGE_SURPRISED
        ]
        
        random_emotion = random.choice(emotions)
        light_matrix.show_image(random_emotion)
        
        await runloop.sleep_ms(1000)

runloop.run(main())

Counting Loop (for loop):

from hub import sound, light_matrix
import runloop

async def main():
    # Countdown from 5 to 1
    for count in range(5, 0, -1):
        light_matrix.write(str(count))
        await sound.beep(500, 300, 80)
        await runloop.sleep_ms(500)
    
    # Blast off!
    light_matrix.write("GO!")
    await sound.beep(800, 1000, 100)
    
    # Repeat an action 3 times
    for i in range(3):
        light_matrix.show_image(light_matrix.IMAGE_HAPPY)
        await runloop.sleep_ms(300)
        light_matrix.clear()
        await runloop.sleep_ms(300)

runloop.run(main())

Conditional Loop (while with condition):

import distance_sensor
import motor_pair
import runloop
from hub import port

async def main():
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    # Move forward until obstacle is detected
    while True:
        distance = distance_sensor.distance(port.C)
        
        # Continue while no obstacle (distance > 200mm or -1)
        if distance > 200 or distance == -1:
            motor_pair.move(motor_pair.PAIR_1, 0, velocity=300)
        else:
            # Stop when obstacle detected
            motor_pair.stop(motor_pair.PAIR_1)
            break  # Exit the loop
            
        await runloop.sleep_ms(50)
    
    print("Obstacle detected! Stopped.")

runloop.run(main())

Loop Types:

  • while True: Runs forever (most common)
  • for i in range(5): Runs 5 times (0,1,2,3,4)
  • while condition: Runs while condition is true
  • break: Exits a loop early

Pro Tip:

Always include await runloop.sleep_ms() in loops to prevent the program from running too fast and allow other tasks to run.

If Statements & Conditions

Make decisions in your code using if, elif, and else statements to create smart robot behaviours.

Basic If Statement:

import color_sensor
import color
from hub import light_matrix, port
import runloop

async def main():
    while True:
        detected_color = color_sensor.color(port.A)
        
        if detected_color == color.RED:
            light_matrix.show_image(light_matrix.IMAGE_ANGRY)
        elif detected_color == color.GREEN:
            light_matrix.show_image(light_matrix.IMAGE_HAPPY)
        elif detected_color == color.BLUE:
            light_matrix.show_image(light_matrix.IMAGE_SAD)
        else:
            light_matrix.show_image(light_matrix.IMAGE_CONFUSED)
            
        await runloop.sleep_ms(100)

runloop.run(main())

Multiple Conditions (and, or):

import distance_sensor
import force_sensor
from hub import light_matrix, sound, port
import runloop

async def main():
    while True:
        distance = distance_sensor.distance(port.A)
        force_pressed = force_sensor.pressed(port.B)
        
        # Multiple conditions with 'and'
        if distance < 100 and distance > 0 and force_pressed:
            light_matrix.show_image(light_matrix.IMAGE_SURPRISED)
            await sound.beep(800, 200, 100)
            
        # Multiple conditions with 'or'
        elif distance < 50 or force_pressed:
            light_matrix.show_image(light_matrix.IMAGE_HAPPY)
            
        # Range checking
        elif distance >= 100 and distance <= 300:
            light_matrix.show_image(light_matrix.IMAGE_ARROW_N)
            
        else:
            light_matrix.clear()
            
        await runloop.sleep_ms(100)

runloop.run(main())

Smart Robot Behaviour:

import color_sensor
import distance_sensor
import color
import motor_pair
from hub import port
import runloop

async def main():
    motor_pair.pair(motor_pair.PAIR_1, port.A, port.B)
    
    while True:
        line_color = color_sensor.color(port.C)
        obstacle_distance = distance_sensor.distance(port.D)
        
        # Priority 1: Avoid obstacles
        if obstacle_distance < 150 and obstacle_distance > 0:
            motor_pair.move(motor_pair.PAIR_1, 100, velocity=200)  # Turn right
            
        # Priority 2: Follow red line
        elif line_color == color.RED:
            motor_pair.move(motor_pair.PAIR_1, 0, velocity=300)    # Go straight
            
        # Priority 3: Search for line
        elif line_color == color.BLACK or line_color == color.UNKNOWN:
            motor_pair.move(motor_pair.PAIR_1, -30, velocity=200)  # Turn left slowly
            
        # Default: Stop
        else:
            motor_pair.stop(motor_pair.PAIR_1)
            
        await runloop.sleep_ms(50)

runloop.run(main())

Comparison Operators:

  • == Equal to
  • != Not equal to
  • < Less than
  • > Greater than
  • <= Less than or equal
  • >= Greater than or equal

Pro Tip:

Put the most important conditions first! In robotics, obstacle avoidance usually comes before other behaviours to keep your robot safe.
